### 🚀 Featured Projects

- 🤖 **[Multi-LLM Collaborative Debate System](https://github.com/NikolozR/YOUR-REPO-LINK)** – A Python-based multi-agent system where three LLMs collaboratively solve complex logic puzzles via a structured pipeline of independent generation, peer review, refinement, and final judgment.

- 🔄 **[Consistent Hashing Simulator](https://github.com/NikolozR/YOUR-REPO-LINK)** – Modeled distributed key redistribution and node churn by designing an interactive Consistent Hashing ring simulation. Achieved $O(\log N)$ lookup efficiency by implementing a custom Binary Search Tree (BST) from scratch.

- 🗣️ **[Natural Language No-SQL Agent](https://github.com/NikolozR/YOUR-REPO-LINK)** – A conversational AI agent capable of translating unstructured English commands into complex, structured SQLite database operations.

- ⚖️ **[Suliko.ge](https://github.com/NikolozR/suliko)** – Scaled an AI-powered legal translation platform to **1,000+ MAU**. Architected the Next.js frontend, integrated a low-latency .NET backend, and engineered a custom rich-text editor with complex state management to sync background AI workers.
